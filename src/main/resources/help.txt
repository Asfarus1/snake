Параметры можно задать из командной строки в формите ИМЯ_ПАРАМЕТРА ЗНАЧЕНИЕ_ПАРАМЕТРА или разместить в той же папке файл с настройками snake.properties

Список параметров:
height - количество ячеек по вертикали
width - количество ячеек по горизонтали
snake.length - начальная а также минимальная длина змейки
snake.sleepDelayMs - количество мс. между ходами змейки
frogsCount - количество лягушек
frog.sleepDelayMs - количество мс. между ходами лягушек
frog.bestDirectionProbably - вероятность с которой лягушка выбирает направление прочь от змейки, может быть от 0.3 до 0.95

help - вызов справки
